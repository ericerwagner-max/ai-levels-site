A field guide

The Levels of AI for Researchers

The same underlying models, used in five increasingly capable ways. As you climb, you hand off more of the doing — and your role shifts from typing every prompt to specifying outcomes and reviewing results.

You do the work AI does the work
💬
Level 1
Chat
Ask & draft — a conversation partner
You drive every turn

Pure back-and-forth with the model. It knows what it was trained on and answers from that. Fastest way to think out loud, but it can't see your files or the live web, so you verify the facts yourself.

· Explain a method or concept · Brainstorm hypotheses · Rewrite a paragraph · Draft an email or abstract
Looks like: Claude.ai, ChatGPT, Gemini — a chat box.
🔎
Level 2
Assisted
Search, retrieve & ground — chat with real sources
You direct, it fetches

Chat plus tools: live web search, document and dataset upload, and citations. Answers are grounded in real material you can check, instead of memory alone. The leap here is evidence — it works from your PDFs and current sources.

· Literature scan with citations · Summarize uploaded papers · Q&A over a dataset or PDF · Pull current guidelines
Looks like: Claude/ChatGPT with web search & file upload, Perplexity, NotebookLM, Copilot in Word/Excel.
🤝
Level 3
Cowork
Do & automate — an agent that takes action
You delegate, it executes

Now it acts. A desktop agent works across your files and apps, runs multi-step tasks, executes code in a sandbox, and produces finished deliverables — documents, spreadsheets, slides, analyses. You hand off a goal; it does the steps and you review the output.

· Run a stats analysis & write it up · Build a manuscript or grant draft · Turn data into a deck or report · Organize & clean a folder
Looks like: Cowork, Claude with computer use, agentic assistants that touch real files and tools.
⌨️
Level 4
Code
Build & scale — reproducible, repeatable work
You specify, it builds

Agentic coding in your terminal or codebase. Instead of one-off answers, it builds the machinery: reproducible analysis pipelines, custom scripts, and tools you can rerun on new data. This is where research becomes scalable and auditable.

· Reproducible analysis pipeline · Custom data-cleaning scripts · Figures generated from code · A small research app or dashboard
Looks like: Claude Code, Cursor, Copilot, agentic coding in an IDE or terminal.
🛰️
Level 5
Autonomous
Orchestrate — agents that run on their own
You set goals, it runs

Custom and background agents built on the API/SDK: scheduled tasks, continuous monitoring, and multi-agent workflows that run without you in the loop each step. Highest autonomy — and the level that most demands guardrails, oversight, and validation.

· Daily literature / alert digests · Scheduled data-pipeline runs · Multi-agent research workflows · AI embedded in your own tools
Looks like: Claude Agent SDK / API, scheduled tasks, orchestration frameworks, custom-built agents.
How to read the ladder: climbing isn't about a "smarter" model — it's the same models given more reach and autonomy. Match the level to the task: chat to think, assisted to ground in evidence, cowork to produce deliverables, code to make it reproducible, autonomous to run continuously. Higher levels save more effort but need more verification — your judgment is the constant at every rung.
